# WebFlex_Clase_1_Entregable

## Entregable: Estructura base de tu Sitio Web

**Objetivo Crear el esqueleto semántico de la página de inicio de tu sitio web personal, sin estilos visuales (eso va en el próximo módulo).**

***Qué entregar Un archivo index.html con la siguiente estructura:***
    
    <header>: nombre/marca y menú de navegación.
    <nav>: lista de enlaces a "Inicio", "Sobre mí", "Proyectos" y "Contacto".
    <main>:
        Sección de presentación con <h1>.
        Sección de biografía breve.
        Sección de servicios o habilidades en lista.
        Al menos una imagen dentro de <figure> con <figcaption> y atributo alt descriptivo.
    <footer>: copyright y enlaces a redes sociales.

***Criterios de aceptación***
   
    Documento válido: <!DOCTYPE html>, <html lang="...">, <head> y <body> presentes.
    Etiquetas semánticas usadas: <header>, <nav>, <main>, <section>, <article>, <footer>. Sin "divitis".
    Jerarquía de títulos sin saltos (<h1> → <h2>, no <h1> → <h4>).
    Imagen con alt descriptivo dentro de <figure>.
    Código sin errores críticos en el Validador W3C.

***Qué evitar***
    
    Usar <h1> o cualquier etiqueta por su apariencia visual; las etiquetas describen significado, no tamaño.
    Omitir el atributo alt: es el error más común de accesibilidad y afecta a usuarios con lectores de pantalla.
